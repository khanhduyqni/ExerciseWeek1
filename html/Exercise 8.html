HTML5 tags to help screen readers (behave like divs)

Header tag, for the top area of the page.
Nav tag, for navigation menus.
Article tag, for main content.
Aside tag, for content that is tangentially related to the main content.
Footer tag, for the bottom area of the page.
Progress tag, to show a progress bar.

A meter tag is similar to a progress tag, but there are differences when using interaction (e.g., click events that get fired).
